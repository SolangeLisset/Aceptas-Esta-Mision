# Aceptar la mision

Un juego web indie, tierno y absurdo donde la mision parece ser sobrevivir minijuegos con completos, palomas y gatos... hasta que se revela la verdadera mision: una invitacion real para salir.

Creado por **SolangeLisset**.

## Idea

El jugador avanza por una serie de minijuegos con humor, pixel art, memes suaves y una interfaz pastel. Al final descubre que todo era una forma menos rara, pero mucho mas memorable, de preguntar:

> ¿Te gustaria salir conmigo a tomar un cafe, ver una pelicula o simplemente pasar un rato juntos?

## Caracteristicas

- React + TypeScript + Vite.
- Tailwind CSS para estilos responsive.
- Framer Motion para transiciones, animaciones y microinteracciones.
- Minijuego de atrapar completos.
- Minijuego de esquivar palomas.
- Minijuego de alimentar al gato.
- Boton falso de rechazo que se escapa y cambia de texto.
- Selector de actividad: cafe, pelicula o paseo.
- Selector de fecha y hora.
- Creditos con resumen final de la salida elegida.
- Modo oscuro.
- Sonidos generados con Web Audio.
- Particulas, confeti y celebracion final.
- Guardado automatico del progreso en `localStorage`.

## Instalacion

